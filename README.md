# DBOT Project (Status: Alpha Version)

The DBOT Project is planned as an open project.  
DBOT is a fixed robot mounted on a tabletop.   
The primary goal of this project is not simply to build a robot,   
but to learn and practice algorithm development and debugging through real-world operation and experimentation.  
  
DBOT is implemented in a bare-metal environment.  
It integrates multiple communication infrastructures,   
including Wi-Fi, Bluetooth, Ethernet (ETH), and Zigbee.  
  
The system also incorporates:  
A camera module  
Dual LCD displays  
Stereo speakers  
Dual microphones  
  
Various processing features are being developed to support these components.  
The ultimate goal of the project is to design and implement a home IoT management protocol.  
  
The project scope includes:  
Mechanical design  
Circuit design    
Firmware development  
GUI development  
  
As I am working on this project independently, progress is relatively slow.   
I plan to dedicate approximately two hours per day to DBOT as a personal hobby project rather  
than a commercial or company-driven initiative.
